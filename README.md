# ADXL335 Part 4 – GitHub Pages

## What it does
- Shows AngleX, AngleY, AngleZ (degrees) from the Arduino
- Live plots all three angles
- 3D cube rotates based on the angles

## Arduino
Upload the Arduino sketch that prints:
at **115200 baud** every ~100 ms.

## Run (GitHub Pages)
1. Go to **Settings → Pages**
2. **Source**: Deploy from a branch
3. **Branch**: main / (root)
4. Save — GitHub gives you a URL like:
   `https://YOURNAME.github.io/adxl335-part4/`

## Use
1. Open the GitHub Pages URL in **Chrome/Edge desktop**
2. Click **Connect Serial**
3. Select your Arduino COM port
